# Landing Page - Café Aroma (Cafeteria Artesanal)

## Descrição do Projeto

Esta é uma landing page desenvolvida para uma cafeteria artesanal fictícia chamada "Café Aroma". O projeto foi criado utilizando **HTML**, **CSS** e **JavaScript básicos**, atendendo aos requisitos de um trabalho acadêmico de front-end.

## Características Técnicas

### HTML Básico
- Estrutura semântica com tags apropriadas (`header`, `section`, `footer`)
- Formulário de contato funcional com validação
- Navegação interna com âncoras
- Organização clara e hierárquica do conteúdo

### CSS Básico
- Estilização responsiva com Grid e Flexbox
- Animações CSS personalizadas (fade-in, slide-up, bounce)
- Efeitos de hover e transições suaves
- Design mobile-first com media queries
- Paleta de cores temática (tons de âmbar e marrom)

### JavaScript Básico
- Menu expansível/retrátil para o cardápio
- Carrossel automático de depoimentos de clientes
- Validação de formulário de contato
- Scroll suave entre seções
- Efeitos de animação ao rolar a página
- Interatividade em botões e elementos

## Funcionalidades Implementadas

1. **Header Fixo**: Navegação sempre visível com links para seções
2. **Hero Section**: Seção principal com call-to-action
3. **Sobre Nós**: História da cafeteria com imagens e valores
4. **Cardápio Interativo**: Menu expansível com categorias e preços
5. **Depoimentos**: Carrossel automático de avaliações de clientes
6. **Formulário de Contato**: Com validação JavaScript e informações da empresa
7. **Footer**: Informações finais e branding

1. Abra o arquivo `index.html` em qualquer navegador web moderno
2. A página é totalmente responsiva e funciona em desktop, tablet e mobile
3. Todas as funcionalidades JavaScript funcionam offline

## Recursos Utilizados

- **Fontes**: Arial (web-safe font)
- **Ícones**: Emojis Unicode para compatibilidade universal
- **Imagens**: Fotos de cafeterias e grãos de café
- **Cores**: Paleta baseada em tons de café (âmbar, marrom, bege)

## Funcionalidades JavaScript

### Interatividade do Menu
- Clique nas categorias do cardápio para expandir/recolher
- Ícones que rotacionam para indicar estado

### Carrossel de Depoimentos
- Rotação automática a cada 4 segundos
- Navegação manual através dos pontos indicadores
- Transições suaves entre depoimentos

### Formulário de Contato
- Validação de campos obrigatórios
- Validação de formato de e-mail
- Feedback visual para o usuário
- Reset automático após envio

### Animações e Efeitos
- Animações de entrada ao carregar a página
- Efeitos de hover em imagens e cards
- Scroll suave entre seções
- Destaque da seção ativa na navegação
