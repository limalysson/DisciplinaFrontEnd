# Simple Weather Application

This is a simple weather application that fetches weather data using the Open-Meteo API. The application allows users to enter a location and view the current weather conditions.

## Project Structure

```
simple-weather-app
├── src
│   ├── index.html        # Main HTML document for the application
│   ├── css
│   │   └── styles.css    # Styles for the application
│   └── js
│       ├── app.js        # Main JavaScript file for handling user interactions
│       └── openMeteo.js  # Functions for interacting with the Open-Meteo API
├── .gitignore            # Files and directories to be ignored by Git
└── README.md             # Documentation for the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in your web browser to view the application.

## Usage

- Enter a location in the input field and click the "Get Weather" button.
- The application will display the current weather conditions for the specified location.

## API Information

This application uses the Open-Meteo API to fetch weather data. Ensure you have an internet connection to retrieve the latest weather information.
